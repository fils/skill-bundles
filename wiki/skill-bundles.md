# Skill Bundles: A Living Catalog

**Primary Publication** — Last major update: 2026-05-23 (Iteration 5)

## TL;DR

Skill bundles are reusable packages that combine agent skills with formal context artifacts (validation, mapping, rules, vocabularies, taxonomies, and ontologies).

## State of Research (Iteration 5)

After five iterations, **Rules & Governance** and **Ontology** layers show the clearest adoption. Formal semantic validation (SHACL) and mapping (SSSOM) remain significantly underrepresented in public skill bundle examples.

See the dedicated state document: [[skill-bundles-research-state]]

## Emerging Patterns

1. Skills + Declarative Context Files (most common)
2. Skills + Ontology (growing)
3. Skills + Rules/Governance (emerging)
4. Skills + SHACL/SSSOM (still rare)
