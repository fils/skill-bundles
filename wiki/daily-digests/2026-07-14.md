## Emerging Patterns

1. **Skill discovery taxonomy now spans domains** — ASPIRE (robotics, code-as-policy + evolutionary search), Memento-Skills (general agents, Read-Write Reflective Learning), Skill1 (RL-based co-evolution), EXIF (Alice-Bob self-play). Each applies a different discovery mechanism to a different domain. The taxonomy of skill discovery approaches is now multi-domain.
2. **Dual-plane security governance** — SkillGuard's dual-plane model (context plane + action plane) is the first to recognize that skills operate across two coupled surfaces. This extends skill bundle security from single-plane (tool-level OR context-level) to dual-plane governance. Combined with SkillFortify (formal supply chain) and NVIDIA-verified (publication), security now covers: publication → supply chain → runtime → behavior.
3. **Formal verification arrives for skills** — SkillFortify brings mathematical proofs (Dolev-Yao attacker model, abstract interpretation, capability confinement) to skill supply chains. This is the first formal (not heuristic) defense framework. The lockfile semantics (deterministic, SHA-256 integrity, trust scores) adapt package management patterns to skills.
4. **Agent-designing agents** — Memento-Skills demonstrates agents autonomously constructing, adapting, and improving task-specific agents through skill evolution. Skills as structured markdown files serve as the sole adaptation mechanism (no LLM parameter updates). The Read-Write cycle is a complete skill lifecycle.
5. **Industry productionization of skill governance** — NVIDIA-verified skills represent the commercial/enterprise layer: catalog → scan → sign → document → distribute. Skill cards centralize trust metadata. Cross-platform SKILL.md ensures portability. This is the publication/distribution counterpart to academic security frameworks.
6. **Frequency decomposition for skill credit assignment** — Skill1's insight that a single task-outcome signal naturally decomposes into low-frequency (selection quality) and high-frequency (distillation) components is a novel governance pattern for unified skill lifecycle management.
