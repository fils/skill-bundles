## Patterns Discovered

1. **Orchestration via Meta-Skill**: `dspy-advanced-workflow` skill activates and sequences other skills — a dependency-ordered bundle pattern that GoS formalizes theoretically.
2. **Constraint Rules as First-Class Context**: Superpowers' Iron Laws and Red Flags show that skills can encode *prohibitions*, not just procedures. This expands the context element taxonomy beyond validation to behavioral constraints.
3. **Dependency Graph for Bundle Composition**: Graph of Skills (GoS) provides the first formal model showing that skill dependencies form DAGs that must be resolved — this explains *why* orchestrator skills work and provides an algorithmic approach.
4. **Paired Evaluation for Bundle Quality**: SkillsBench introduces the delta metric (with-skills vs without-skills) — the first quantitative way to assess individual bundle effectiveness.
5. **Security Ecosystem Maturity**: Mondoo reveals 25%+ vulnerability rate + 14 attack patterns — security context elements (scanning, signing, verification) are the most critical gap.
