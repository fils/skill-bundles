3. **Implicit Bundle Discovery**: The awesome-openclaw-skills catalog reveals that bundles can be *discovered* by grouping skills across categories (4+ security skills = implicit security bundle), not just explicitly declared.
4. **Agent Context as Documentation**: OxO2's CLAUDE.md + CONTEXT.md + ADR pattern shows that agent instructions can be treated as living documentation — a formal context artifact that compounds with code changes.
