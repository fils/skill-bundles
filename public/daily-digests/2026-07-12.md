## Emerging patterns

1. **Skill optimization triad converging** — three distinct approaches to optimizing skill bundles now exist: SkillMOO (search-based/NSGA-II), CoEvoSkills (evolution-based), SkillReducer (content reduction/delta debugging). Each addresses a different axis — configuration, content+verifier, and token efficiency — and could compose.
2. **Less-is-more effect quantified** — SkillReducer shows that removing non-essential skill content actually *improves* functional quality by 2.8%, not just preserves it. This challenges the "more context = better" assumption. Progressive disclosure (load on demand) is the architectural pattern.
3. **Skill composition as core capability** — SkillCraft validates that agents composing tools into reusable skills is a *core capability* that strongly correlates with success. 80% token savings from skill caching/reuse. This generalizes the skill bundle thesis from human-authored to agent-authored.
4. **Hierarchical security validation maturing** — SkillSieve's three-layer triage (heuristic→LLM→LLM jury) with $0.006/skill cost and F1=0.920 is production-ready. Cross-ecosystem portability (ClawHub→Feishu) proven. The LLM jury (3-LLM voting + debate) is a novel governance pattern.
5. **Skills + ontologies integrated** — AI-KM closes the loop: knowledge → ontology → skills → knowledge feedback. NL extraction lowers the barrier. Conversation-invokable skills + autonomous planning mode for novel tasks. First concrete implementation of the skill bundle thesis with ontology as the context element.
6. **Marketplace consolidation predicted** — 8 marketplaces → 3-4 within a year. Security scanning and creator payments becoming table stakes. MCP-based access replacing download-based distribution.
