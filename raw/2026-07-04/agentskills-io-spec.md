# agentskills.io — Agent Skills Specification

**Source:** https://agentskills.io/ (home + specification)
**Date ingested:** 2026-07-04
**Type:** Open standard specification

**Key content:**
- Lightweight, open format for extending AI agents with specialized knowledge and workflows.
- Core: folder containing SKILL.md (YAML frontmatter + instructions).
- Progressive disclosure: Discovery → Activation → Execution.
- Supports scripts/, references/, assets/.
- Adopted by GitHub Copilot, Claude Code, Cursor, Codex, Gemini CLI, Tabnine, Snowflake Cortex, and many others.
- Open development on GitHub (agentskills/agentskills).

**Context elements present:**
- Formal specification document
- Client showcase matrix
- Quickstart and creation guides
- Resource extension rules (.md, .json, .yaml, etc.)

**Relevance to skill bundles:** The canonical definition of the skill bundle format itself. Every other example in this catalog is an instance of this standard.
