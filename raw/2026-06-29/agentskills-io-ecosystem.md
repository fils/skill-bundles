# agentskills.io + Ecosystem (Agent Skills Standard)

**Sources:**
- https://agentskills.io/ (official site)
- GitHub topics: agent-skills (multiple repos updated June 2026)
- VS Code / GitHub Copilot / Claude docs on Agent Skills
- VoltAgent/awesome-agent-skills, anthropics/skills, etc.

**Key Ideas:**
- Open standard for Agent Skills (originally from Anthropic, now ecosystem-wide).
- Skills = folders of instructions, scripts, resources loaded dynamically by agents.
- Supported across Claude Code, GitHub Copilot, Codex, Gemini CLI, Cursor, etc.
- Growing catalog of official + community bundles (hundreds of skills in popular repos).
- GitHub CLI now has `gh skill` command for discovery/install.

**Composition Notes:**
- Skills are portable, composable packages.
- Many repos act as "skill bundles" or catalogs.

**Relevance:** Foundational for the entire skill-bundles research domain.
