# addyosmani/agent-skills

**Source:** https://github.com/addyosmani/agent-skills (updated ~8 hours ago as of 2026-06-29)
**Type:** GitHub repository — Production-grade engineering skills for AI coding agents
**Key Ideas:**
- 24 skills total (23 lifecycle skills + 1 meta-skill "using-agent-skills").
- Skills encode workflows, quality gates, and best practices that senior engineers use when building software.
- Designed as installable skill bundles for Claude Code, Codex, Cursor, Gemini CLI, etc.
- Emphasizes senior-engineer scaffolding that LLMs lack out of the box.
- MIT license; intended for teams and tools.

**Composition Notes:**
- Skills are modular Markdown files with frontmatter.
- Lifecycle coverage: planning → implementation → review → deployment.
- Strong example of a "skill bundle" that pairs agent instructions with engineering process guardrails.

**Relevance to Skill Bundles:** Prime example of a curated, production-oriented skill bundle without explicit SHACL/SSSOM in the repo itself, but demonstrates the packaging pattern the catalog tracks.
