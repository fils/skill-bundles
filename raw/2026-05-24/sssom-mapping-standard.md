# SSSOM — Simple Standard for Sharing Ontological Mappings

**Source:** https://github.com/mapping-commons/sssom/
**Date Accessed:** 2026-05-24
**Author:** Mapping Commons

## Key Ideas
- SSSOM provides a TSV-based representation for ontology term mappings
- Comprehensive metadata model for mapping provenance
- Python toolkit available: sssom-py (s/sssom-py)
- Standardization: unified format for representing semantic/ontological mappings
- Designed for interoperability between different systems and organizations

## Bundle Context Elements
- **SSSOM Mapping:** Core specification — TSV format for ontology mappings
- **Python Toolkit:** sssom-py CLI and library for working with SSSOM files
- **Metadata Model:** Provenance tracking for mapping relationships

## Relevance to Skill Bundles
SSSOM is the formal standard for ontology term mappings. An agent skill that uses SSSOM would:
1. Read mapping files between different ontologies
2. Use the mapping to translate queries/knowledge across schema boundaries
3. Provide a formal "translation layer" context element for skills working with multiple data sources
