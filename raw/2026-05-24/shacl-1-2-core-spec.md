## Key Ideas
- SHACL 1.2 is the latest evolution of the Shapes Constraint Language W3C standard
- Validates RDF graphs against sets of conditions (shapes)
- Key for skill bundle validation when skills depend on structured knowledge graphs
- Provides structural validation: ensures data conforms to expected schemas

## Relevance to Skill Bundles
- SHACL shapes can validate skill input/output contracts
- When skills consume or produce structured data, SHACL provides the validation layer
- Core component for the "Validation" context element category in bundle taxonomy
