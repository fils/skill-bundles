# Agent Skills Overview - agentskills.io

**Source:** https://agentskills.io/home (2026-07-01)
**Type:** Specification / Standard

## Key Ideas
- Agent Skills: lightweight, open format for extending AI agent capabilities with specialized knowledge and workflows.
- Core: folder containing SKILL.md (metadata + instructions) + optional scripts/, references/, assets/.
- Progressive disclosure: Discovery (name+desc), Activation (full SKILL.md), Execution.
- Supported by 20+ platforms including Claude Code, GitHub Copilot, Cursor, OpenAI Codex, Gemini CLI, etc.
- Open standard originally from Anthropic, now at agentskills.io; GitHub org agentskills/agentskills.

## Quotes / Highlights
- "Skills solve this by packaging procedural knowledge and company-, team-, and user-specific context into portable, version-controlled folders that agents load on demand."
- "Agents load skills through progressive disclosure, in three stages."

## Relevance to Skill Bundles
Directly defines the core mechanism for bundling skills with supporting context (scripts, references, templates). Matches the research focus on skill bundles + formal context elements.
