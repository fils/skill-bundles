# SHACL 1.2 Core Specification (W3C)

**Source:** https://www.w3.org/TR/shacl12-core/
**Date ingested:** 2026-07-03
**Type:** W3C specification

## Key Ideas
- SHACL 1.2 extends SHACL for validation, inferencing, modeling domains, generating ontologies to inform other agents.
- New properties in 1.2: sh:intent, sh:agentInstruction, sh:codeIdentifier.
- Explicit support for agent instructions via sh:agentInstruction.
- Used in complex agents alongside OWL for constraint gatekeeping and rule-based decisions.

## Quotes / Highlights
- "SHACL may be used for a variety of purposes such as validating, inferencing, modeling domains, generating ontologies to inform other agents..."
- sh:agentInstruction section added for agent-specific instructions.

## Relevance to Skill Bundles
Direct evidence of SHACL as a formal context element for agent skills (validation + agent instructions). SHACL shapes can validate skill inputs/outputs and provide guardrails. Key for "skills + SHACL" bundle pattern.
