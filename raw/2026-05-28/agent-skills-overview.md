# Agent Skills Overview (2026-05-28)

Source: https://agentskills.io/ and related Anthropic announcements

## Summary
Agent Skills represent a major new standard for packaging agent capabilities as composable, version-controlled folders containing SKILL.md with YAML frontmatter (name, description) and optional scripts, references, assets.

This directly aligns with "skill bundles" concept: skills + supporting context (instructions, code, schemas).

## Key Elements
- Progressive disclosure for efficient context use
- Supported across many platforms (Claude, Codex, etc.)
- Open standard at agentskills.io
- Examples include document processing skills with validation-like structures

## Relevance to Skill Bundles
This is a prime example of skill bundles. Future work should look for bundles that add SHACL validation, SSSOM mappings, ontologies to these skills.
