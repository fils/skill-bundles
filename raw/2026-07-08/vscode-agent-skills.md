# Use Agent Skills in VS Code

**Source:** https://code.visualstudio.com/docs/agent-customization/agent-skills
**Date ingested:** 2026-07-08
**Key ideas:** Agent Skills in VS Code / GitHub Copilot: folders with SKILL.md + resources/scripts. Project skills in .github/skills/, personal in ~/.copilot/skills/. Portable across agents. Vs custom instructions: skills include scripts/examples, on-demand loading, open standard.

**Quotes:**
- "Agent Skills are folders of instructions, scripts, and resources that GitHub Copilot can load when relevant to perform specialized tasks."
- "Skills enable specialized capabilities and workflows that can include scripts, examples, and other resources."
- "Agent Skills is an open standard (agentskills.io) that works across multiple AI agents"

**Relevance to skill bundles:** VS Code support shows practical bundling; skills often bundled with validation (SHACL) or mapping (SSSOM) context for Copilot/GitHub ecosystem.
