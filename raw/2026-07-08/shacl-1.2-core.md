# SHACL 1.2 Core (W3C)

**Source:** https://www.w3.org/TR/shacl12-core/
**Date ingested:** 2026-07-08
**Key ideas:** SHACL (Shapes Constraint Language) 1.2 for describing RDF graph structures, validation, inferencing, modeling domains, generating ontologies to inform other agents. New in 1.2: sh:agentInstruction, sh:intent, sh:codeIdentifier. Used for agent decision-making, policy compliance, data validation in agentic workflows.

**Quotes:**
- "SHACL can be used to define classes and the properties that instances of these classes can have."
- "validating, inferencing, modeling domains, generating ontologies to inform other agents, building user interfaces, generating code, and integrating data."
- Sections on sh:agentInstruction highlight direct relevance to agent skills.

**Relevance to skill bundles:** SHACL shapes provide formal validation/context for skills; bundles often pair skills with SHACL for guardrails (e.g., OntoSkills, constraint checking).
