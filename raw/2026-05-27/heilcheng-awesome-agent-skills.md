# heilcheng/awesome-agent-skills

**Source:** https://github.com/heilcheng/awesome-agent-skills (live at agent-skill.co)
**Date ingested:** 2026-05-27
**Signal:** High (curated directory + marketplace of agent skills across platforms, with quality standards and discovery tools)

**Summary:** Community-curated list of simple text files (SKILL.md) that teach AI assistants specific tasks on demand. Focuses on real, high-quality skills rather than bulk-generated. Includes official directories from Anthropic, OpenAI, Google, Cloudflare, etc., plus community skills. Companion site agent-skill.co and npx skills CLI for discovery/install.

**Skills included:** Hundreds across categories (AI Platforms, Cloud & Infrastructure, Developer Tools, Google Ecosystem, Business/Productivity/Marketing, Security & Web Intelligence, Community). Examples: Anthropic (docx, pptx, mcp-builder), OpenAI (playwright, imagegen, linear), VoltAgent, Stripe, Notion, Trail of Bits, etc. Many implicit bundles (e.g., marketing skills, PM skills by Dean Peters 46 skills, Pawel Huryn 65 skills).

**Context elements:**
- SKILL.md format (clear, actionable, ≤500 lines, examples, trade-offs)
- SkillsMP Marketplace, skills.sh Leaderboard
- npx skills CLI (find, add, list, check, remove)
- Official skill directories per platform
- Quality standards (unambiguous, focused scope, working examples, documented limitations)

**Composition notes:** Skills are standalone but grouped into categories and official directories. Discovery via marketplaces and CLI. Supports loading via /skills add or paste URL. Many skills form implicit bundles (e.g., full marketing stack, security stack).

**Reproducibility:** High — simple markdown, cross-platform compatible, CLI tooling for management.

**Related patterns:** Distribution (Plugins, Marketplaces), Enterprise platforms (Microsoft, Spring AI via inclusion), Security governance (Trail of Bits, Sentry).
