**Date ingested:** 2026-06-27

## Key Developments
- GitHub CLI now supports `gh skill install` for agent skills across Claude Code, Cursor, Codex, Gemini CLI.
- Agent Skills specification (agentskills.io) is the open standard.
- Major curated collections emerging: VoltAgent awesome list, alirezarezvani/claude-skills (345 skills), agent-skills.cc directory.
- VS Code experimental support in Copilot for .github/skills/, .claude/skills/ etc.

## Relevance to Skill Bundles
Explosion of portable skill libraries + official platform support. Context elements (validation, mapping) still lagging behind the skills themselves — opportunity for SHACL/SSSOM integration.
