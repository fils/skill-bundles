## Key Ideas
- SSSOM TSV + metadata standard for ontology mappings, with creator, mapping_date, curation_rule fields.
- Python toolkit (sssom-py) for parsing, conversion to OWL/JSON-LD.
- Integration with workflow provenance (PROV to Workflow Run Crate mappings).
- New browser/tools for logically sound crosswalks.

## Relevance to Skill Bundles
SSSOM remains the canonical context element for mapping/taxonomy layers in skill bundles. Recent toolkit releases and arXiv work strengthen its role in agentic curation and ontology skills.
