# SHACL 1.2 Core (W3C)

**Source:** https://www.w3.org/TR/shacl12-core/
**Date ingested:** 2026-06-27
**Type:** Specification update

## Key Ideas
- SHACL 1.2 extends core with new properties for LLM and agent input (see GitHub issue #725 in data-shapes).
- Supports defining classes, properties, and constraints more expressively.
- Used as constraint gatekeeper in agentic systems alongside OWL for inferencing.

## Quotes
"SHACL can be used to define classes and the properties that instances of these classes can have."

## Relevance to Skill Bundles
Direct bridge between SHACL validation and agent/LLM skills. New property for capturing LLM/agent input makes SHACL a first-class context element for agent skills.
