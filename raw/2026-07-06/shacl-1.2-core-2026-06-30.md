# SHACL 1.2 Core (WD 2026-06-30)

**Source:** https://www.w3.org/TR/shacl12-core/
**Date ingested:** 2026-07-06
**Key ideas:**
- SHACL 1.2 Core extends SHACL for validating, inferencing, modeling domains, generating ontologies to inform other agents, building UIs, generating code, integrating data.
- New features include sh:intent, sh:agentInstruction, sh:codeIdentifier (sections 8.2-8.4).
- Supports agentic workflows by generating ontologies and instructions for agents.
- SHACL shapes graphs describe data graphs; used for constraint checking of skills.

**Quotes:**
- "SHACL may be used for a variety of purposes such as validating, inferencing, modeling domains, generating ontologies to inform other agents, building user interfaces, generating code, and integrating data."
- Mentions sh:agentInstruction for agent guidance.

**Relevance to skill bundles:** Direct support for bundling SHACL shapes with agent skills for validation and instruction.
