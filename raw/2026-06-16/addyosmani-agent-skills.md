**Type:** GitHub repository - Production-grade engineering skills for AI coding agents

## Key Ideas
- Skills encode senior-engineer workflows, quality gates, and best practices for the full development lifecycle (DEFINE → PLAN → BUILD → VERIFY → REVIEW → SHIP).
- 24 skills organized by lifecycle stage + 4 agent personas (code-reviewer, test-engineer, security-auditor, web-performance-auditor).
- Every SKILL.md has strict structure: Frontmatter, Overview, When to Use, Process, Rationalizations table, Red Flags, Verification.
- Commands like /spec, /plan, /build auto, /test, /review, /ship enforce discipline.
- Focus on reproducibility, test-driven commits, progressive disclosure.
