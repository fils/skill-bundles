## Key Ideas
- Skills are folders with SKILL.md + optional scripts/resources that Claude loads dynamically.
- Includes example skills across categories + production document skills (docx, pdf, pptx, xlsx).
- Points to agentskills.io specification.
- Template for creating skills with YAML frontmatter (name, description).
