## Summary

Production-grade engineering skills for AI coding agents. Encodes senior engineer workflows, quality gates, and best practices across the full software lifecycle (Define → Plan → Build → Verify → Review → Ship).

**24 skills** organized by phase, each with:
- Overview, When to Use, Process (step-by-step), Rationalizations (anti-excuse tables), Red Flags, Verification (evidence requirements).
- Uses slash commands (/spec, /plan, /build, /test, /review, /ship, etc.).
- Includes 4 specialist personas (code-reviewer, test-engineer, security-auditor, web-performance-auditor).

**Context elements included:**
- Testing patterns, security checklist, performance checklist, accessibility checklist.
- Strong emphasis on verification and evidence-based workflows.

## Relevance to Skill Bundles

This is a canonical example of a high-quality, production-oriented skill bundle. It demonstrates how to package procedural knowledge with explicit guardrails (verification steps, red flags) without requiring external SHACL/SSSOM in this instance, but the structure itself serves as a model for composable bundles.
