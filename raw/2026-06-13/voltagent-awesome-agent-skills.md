**Category:** Curated Catalog / Awesome List

## Summary

Curated collection of 1000+ agent skills from official dev teams and community. Explicitly positions itself as high-quality (not AI-generated bulk).

**Key Features:**
- Compatibility matrix for Claude Code, Codex, Gemini CLI, Cursor, GitHub Copilot, etc.
- Official skills from Anthropic, Google, Microsoft, Stripe, Vercel, etc.
- Community categories: Vector DBs, Marketing, PM, Security (Trail of Bits), etc.
- Strict quality criteria for submissions (third-person descriptions, progressive disclosure, no absolute paths).

**Relevance to Skill Bundles**

This is the central discovery hub for agent skills. It functions as a meta-bundle / catalog that points to hundreds of individual skill bundles. Excellent source for tracking emerging patterns in how skills are organized and quality-controlled at scale.
