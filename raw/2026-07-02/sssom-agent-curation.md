# SSSOM — Agent & Curation Rule Support

**Source:** https://mapping-commons.github.io/sssom/dev/ and GitHub
**Date ingested:** 2026-07-02
**Type:** Specification + tooling update

**Key Ideas:**
- SSSOM mapping sets include metadata for creator (agent), curation_rule, curation_rule_text.
- Supports agent-driven mapping curation.
- TSV-based, OWL translation, used in bioinformatics/ontology mapping.

**Quotes/Highlights:**
- "The creator is the agent that put the mapping in its published form"
- "curation_rule: A curation rule is a (potentially) complex condition executed by an agent"
- Python library sssom (latest May 2026) for parsing/operations.

**Context Elements:** SSSOM mappings with explicit agent/curator metadata and rules.

**Relevance to Skill Bundles:** Strong example of mapping context (SSSOM) with agent primitives — complements SHACL for bundle validation + mapping.
