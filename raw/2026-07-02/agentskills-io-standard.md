# agentskills.io Standard & Ecosystem

**Source:** https://agentskills.io/ (via GitHub topics, VS Code docs, GitHub Copilot docs)
**Date ingested:** 2026-07-02
**Type:** Open standard specification + ecosystem

**Key Ideas:**
- Open standard for Agent Skills: folders of instructions, scripts, and resources.
- Works across GitHub Copilot, Claude Code, Codex CLI, Gemini CLI, etc.
- Project skills stored in .github/skills/, .claude/skills/, .agents/skills/.
- Progressive disclosure, SKILL.md format.

**Quotes/Highlights:**
- "Agent Skills is an open standard that works across multiple AI agents..."
- "Skills allow Copilot to perform specialized tasks."
- GitHub Copilot supports project skills and community collections (awesome-copilot).

**Context Elements:** Standard format (SKILL.md), ecosystem of 1000+ skills, validation via community repos (agent-skills-eval, etc.).

**Relevance to Skill Bundles:** Foundational spec for all skill bundles. Defines the packaging format that SHACL/SSSOM/rules augment.
