# agentskills.io Overview

**Source:** https://agentskills.io/home
**Date ingested:** 2026-06-03

Official spec for Agent Skills: lightweight open format. Folder with SKILL.md + optional scripts/references/assets.

Progressive disclosure: Discovery → Activation → Execution.

Supported by many clients including Claude Code, GitHub Copilot, Cursor, etc.

GitHub org: https://github.com/agentskills/agentskills

This provides the foundational spec and taxonomy for skill bundles. No SHACL/SSSOM in core, but extensible with references/ for context artifacts.
