# heilcheng/awesome-agent-skills

**Source**: https://github.com/heilcheng/awesome-agent-skills (live directory at agent-skill.co)
**Date ingested**: 2026-06-11
**Type**: Curated catalog / directory of Agent Skills

## Summary
Community-curated index of hundreds of Agent Skills (SKILL.md files) across providers (Anthropic, OpenAI Codex, Google Gemini, GitHub Copilot, etc.). Includes discovery tools like SkillsMP Marketplace and skills.sh leaderboard.

## Key Context Elements
- Standardized SKILL.md template with frontmatter
- Categories: Vector DBs, Marketing, Productivity, Dev/Testing, Context Engineering, Security
- Provider-specific directories and examples
- Quality standards: <500 lines, actionable, tested, trade-offs documented

## Composition Notes
This is a meta-bundle: a living catalog that aggregates many individual skill bundles. Demonstrates ecosystem growth around the Agent Skills open standard. No formal SHACL/SSSOM but strong community-driven taxonomy and quality gates.

## Reproducibility
Medium-High — GitHub-based, many public repos. Directory itself is a useful research artifact for discovering composition patterns.
