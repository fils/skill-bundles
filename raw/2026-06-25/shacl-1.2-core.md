# SHACL 1.2 Core

**Source:** https://www.w3.org/TR/shacl12-core/  
**Date:** 2026-06-02 (published)  
**Ingested:** 2026-06-25

## Key Ideas
- SHACL 1.2 extends the Shapes Constraint Language with new properties for agent and LLM integration: `sh:intent`, `sh:agentInstruction`, `sh:codeIdentifier`.
- Supports validating, inferencing, modeling domains, generating ontologies to inform other agents, building user interfaces, generating code, and integrating data.
- SHACL can define classes and structure of RDF graphs, used as constraint gatekeeper alongside OWL inferencing in agentic systems.

## Notable Quotes / Features
- "validating, inferencing, modeling domains, generating ontologies to inform other agents, building user interfaces, generating code, and integrating data."
- New sections on `sh:agentInstruction` and `sh:codeIdentifier` explicitly target LLM/agent workflows.
- SHACL 1.2 Core specification emphasizes machine-readable shapes that can guide agent behavior and validation.

## Relevance to Skill Bundles
Direct support for bundling SHACL shapes with agent skills for validation, instruction, and code generation in agentic AI. Strong signal for SHACL + agent skills bundles.
